在centos7上安装python3

Written by yyc on November 8, 2019 in Linux

centos7上默认安装的python版本是2.7， 实在是太旧了，需要另外安装一个python3，过去常用的方法是源码编译安装，然后使用软链接到python3，有些麻烦，本文使用yum源scl来安装，方便快捷。

  • 安装scl：

    sudo yum install centos-release-scl

  • 检查scl中支持的python3版本：这个地址

  • 安装选定版本的python3：

    sudo yum install rh-python36

  • bash中默认使用新版python：

    scl enable rh-python36 bash

  • 安装开发工具：

    sudo yum groupinstall 'Development Tools'

  • 检查下python、pip版本，创建虚拟环境：

    python --version
pip --version
python -m venv venv
source venv/bin/activate
deactivate
